Wörter web application

What is this?

This is yet another application that helps you learning foreign language. This app was created for German, but as a matter of fact you can use it for any language.

What does "wörter" mean?

Wörter is "words" in German.

Why should I use this app instead of another one?

It allows you to create your own sets of words. You learn only the words you want to learn. It is not a standalone learning language application. It's just a tool that may help you learn language more effectively.
